# Lumina E-Commerce Platform

A premium, full-stack e-commerce platform built with Flask (Backend) and Vanilla JS/CSS (Frontend).

## Features
- **Modern UI**: SPA-like experience with smooth animations.
- **Secure Auth**: JWT-based authentication.
- **Product Catalog**: Dynamic search and category filtering.
- **Cart & Checkout**: Real-time stock management and order processing.
- **Admin Dashboard**: Business analytics and order tracking.

## Getting Started
1. **Backend**: `python -m backend.app`
2. **Frontend**: `python -m http.server 8000 --directory frontend`
